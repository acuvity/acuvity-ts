# Extraction

Represents the extracted information to log.

## Example Usage

```typescript
import { Extraction } from "acuvity/models/components";

let value: Extraction = {
  categories: [
    {
      group: "image",
      type: "png",
    },
  ],
  confidentiality: 0.9,
  modalities: [
    {
      group: "image",
      type: "png",
    },
  ],
  relevance: 0.9,
};
```
