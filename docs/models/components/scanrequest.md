# Scanrequest

This is a scan request.

## Example Usage

```typescript
import { Scanrequest } from "@acuvity/acuvity/models/components";

let value: Scanrequest = {
  bypassHash: "Alice",
  user: {
    claims: [
      "@org=acuvity.ai",
      "given_name=John",
      "family_name=Doe",
    ],
    name: "Alice",
  },
};
```
